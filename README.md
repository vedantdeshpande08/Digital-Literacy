## Project Overview
This project was completed as part of the Digital Ambassador program. It focuses on developing essential digital skills such as online communication, coding practice, collaboration using cloud tools, and awareness of cyber safety. The project includes multiple tasks that help in understanding both technical and responsible use of digital platforms.

## Module Summaries
### Task 1: Digital Literacy Presentation
Created a presentation using Canva to explain digital literacy, online safety, and responsible internet usage.

### Task 2: Portfolio Setup
Built professional profiles on LinkedIn and GitHub to establish an online presence and showcase skills.

### Task 3: Coding & Collaboration Platforms
Solved a beginner problem on HackerRank and created a Digital Literacy Quiz using Google Forms linked with Google Sheets.

### Task 4: Email Etiquette & Social Media Awareness
Wrote professional emails and created a checklist for responsible social media usage.

### Task 5: Cybercrime Awareness
Studied UPI fraud, created a case study, and prepared a prevention checklist for online safety.
